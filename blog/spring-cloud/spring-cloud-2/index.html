spring cloud 族谱

March 24, 2018

功能简介来自官网 整理人：无心

总结下 ： 可以说 spring cloud 服务中 包括各种spring 基础服务也有了很多更新

有RPC 框架的 注册 netflix
有 关于配置的 远程仓库配置 config
有 关于消息 bus stream
有关于app的 for android (提供RestTemplate)
有关任务调度 的 task
有关于 shell 编程
有关于安全方面的
有关于系统拦截 zuul
等等。


那么看名字可以分为两种
1、Spring * --- spring 为了完善"应用"增加的功能机制  例如 spring security 、spring vault、等等
2、Spring Cloud * ---spring cloud * 是在 微服务中 为了"微服务系统"更方便的集成 对各个功能模块进行封装的结果、
